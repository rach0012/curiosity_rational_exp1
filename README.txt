Code and data for Experiment 1 for our 2020 psych review paper: "Reconciling Novelty and Complexity through a Rational Analysis of Curiosity"

Curiosity3.json is the experiment data (of Exp 1)

analysis_data_all_paper.m is the main file that does plotting and analysis of Exp 1 (for both phases)

analysis_data_all_binned_paper.m is the file that does individual-level analysis of Exp 1 (the one reported in the Appendix)

Note: before running the codes, remember to include the folder `functions' in the file path.
